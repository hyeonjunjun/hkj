# Elite Portfolio Research: What Separates Top 1% Design Engineers

## The Core Finding

Every unforgettable portfolio shares one trait: **the site itself demonstrates the skill it claims.** Rauno's OS proves he understands interaction. Paco's radical minimalism proves he values performance. Dennis's cursor-following imagery proves he builds GSAP experiences. The site IS the portfolio piece.

## What the Top 5 All Have in Common

1. **A singular concept that IS the portfolio** — not "hero + grid"
2. **Typography as primary design material** — 2 families max, hierarchy through weight/opacity
3. **Restraint in one dimension enables excess in another** — pick your axis
4. **The site demonstrates the skill it claims** — don't tell, show
5. **No standard project grid on homepage** — text links, paginated images, hover-reveals, or cinematic scroll

## The #1 Differentiator

**A point of view expressed through the medium itself.** Good portfolios display work. Unforgettable portfolios express philosophy through HOW they present work.

## Overused (Avoid)

- Dennis Snellenberg clone (smooth scroll + cursor images + GSAP text reveals)
- "Hey, I'm [Name]" hero with waving emoji
- Bento grid layouts
- Gradient mesh backgrounds
- "Selected works" with 2-3 project cards
